== README

* Ruby version: 2.2.3

* System dependencies

* Configuration

* Database creation:
  * `rake db:create`

* Database initialization:
  * `rake db:migrate`

* How to run the test suite:
  * `rake spec`
  * `rake jasmine`

* Deployment instructions:
  * `git push heroku master`
  * `git push staging master`
  * production: https://git.heroku.com/donut-ux.git
  * staging: https://git.heroku.com/staging-donut-ux.git
